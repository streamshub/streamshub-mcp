# 4.4 Resource Templates for New CRDs

## Context

Currently 5 resource templates exist (Kafka status/topology, NodePool status, Topic status, Operator status). Need resource templates for all new CRDs added in Phase 2, plus resource subscriptions for live updates.

## New Resource Templates

| Resource | URI Pattern |
|----------|-------------|
| KafkaConnect status | `strimzi://kafka.strimzi.io/namespaces/{namespace}/kafkaconnects/{name}/status` |
| KafkaConnector status | `strimzi://kafka.strimzi.io/namespaces/{namespace}/kafkaconnectors/{name}/status` |
| KafkaUser status | `strimzi://kafka.strimzi.io/namespaces/{namespace}/kafkausers/{name}/status` |
| KafkaRebalance status | `strimzi://kafka.strimzi.io/namespaces/{namespace}/kafkarebalances/{name}/status` |
| KafkaMirrorMaker2 status | `strimzi://kafka.strimzi.io/namespaces/{namespace}/kafkamirrormaker2s/{name}/status` |
| KafkaBridge status | `strimzi://kafka.strimzi.io/namespaces/{namespace}/kafkabridges/{name}/status` |
