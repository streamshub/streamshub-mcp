# StreamsHub MCP

A collection of MCP (Model Context Protocol) servers for the streaming portfolio, built with Quarkus.

## MCP Servers

| Module | Description |
|--------|-------------|
| [strimzi-mcp](strimzi-mcp/) | Strimzi Kafka operator devops tools for AI assistants |

## Build

```bash
mvn compile       # compile all modules
mvn test          # run tests
```
